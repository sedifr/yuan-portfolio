## ❌ 如果遇到问题

### 问题 1: git push 失败

**错误信息**: `Permission denied` 或 `Authentication failed`

**解决方法**:
1. 确认你已经登录 GitHub
2. 或者使用 GitHub Desktop 上传

### 问题 2: Vercel 部署失败

**错误信息**: 部署日志显示红色错误

**解决方法**:
1. 检查 GitHub 仓库是否有所有文件
2. 检查环境变量是否正确填写
3. 在 Vercel 点击 "Redeploy" 重试

### 问题 3: 保存时显示"GitHub 未配置"

**解决方法**:
1. 检查 Vercel → Settings → Environment Variables
2. 确认 4 个变量都正确填写
3. 注意 `GITHUB_TOKEN` 必须是完整的 token (以 ghp_ 开头)
4. 修改后点击 "Redeploy"

### 问题 4: 网站显示但图片丢失

**解决方法**:
1. 检查 GitHub 仓库的 `assets/images/` 文件夹
2. 确保所有图片都已上传
3. 如果缺失,手动上传图片到 GitHub
4. Vercel 会自动重新部署
