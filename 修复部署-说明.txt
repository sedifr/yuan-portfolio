🎯 修复了什么?
========================================

问题原因:
应用卡片有 reveal 动画效果(初始 opacity: 0),
但缺少触发动画的 JavaScript 代码,导致一直保持透明。

解决方案:
在 loadApps() 函数中添加了自动触发代码,
应用加载后会自动添加 'visible' 类,触发显示动画。

效果:
✨ 4 个应用会依次渐入显示,间隔 100ms
✨ 优雅的动画效果
✨ 完全自动,无需手动操作


📝 技术细节
========================================

修改文件: data-loader.js (第 214-223 行)

添加的代码:
```javascript
// 触发 reveal 动画
setTimeout(() => {
  const appCards = document.querySelectorAll('.app-card.reveal');
  appCards.forEach((card, index) => {
    setTimeout(() => {
      card.classList.add('visible');
    }, index * 100);
  });
}, 100);
```
